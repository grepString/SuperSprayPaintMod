Installation Guide

1. Install UE4SS

  • Download experimental-latest version of UE4SS from GitHub
  • Place the ue4ss and dwmapi.dll folder in:
    Drive Beyond Horizons\DriveBeyondHorizons\Binaries\Win64

2. Install the Mod

  • Option 1: Download the mod from GitHub
  • Option 2: Download the mod from Nexus Mods
  • Place the SuperSprayPaintMod folder in:
    Drive Beyond Horizons\DriveBeyondHorizons\Binaries\Win64\ue4ss\Mods\

3. Mod Configuration (Optional)

The mod comes pre-enabled with an enabled.txt file. No additional setup is required.

If you prefer to manage mods through configuration files, you can:

  • Delete the enabled.txt file and add the mod to your configuration:
    • mods.txt:
SuperSprayPaintMod : 1

mods.json:
{
  "SuperSprayPaintMod": true
}

Troubleshooting

  • Check the UE4SS console for error messages (requires zDEV-UE4SS install)
  • Verify the mod is properly enabled (either with enabled.txt or in mods.txt/mods.json)
  • For paint can spawn issues, ensure you're on flat ground (Gas Stations and Repair Shops work best)
Known Issues:
  • Placing spawned cans in hotbar slots will overwrite the modded can - simply spawn a new one
  • The game may crash when exiting via the in-game menu after spawning cans. While annoying, this is harmless and your progress will be saved

Multiplayer Notes

  • All players should have the mod installed
  • Host-spawned cans have priority for paint application
  • Host can spawn preset color cans for other players
  • Non-host players cannot change colors of host-spawned cans
